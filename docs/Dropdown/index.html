Dropdown 下拉菜单

向下弹出的列表。

何时使用

当页面上的操作命令过多时，用此组件可以收纳操作元素。点击或移入触点，会出现一个下拉菜单。可在列表中进行选择，并执行相应的命令。

  1. 用于收罗一组命令操作。
  2. Select 用于选择，而 Dropdown 是命令集合。

基本用法

包含 disabled、Divider、danger 的使用

<Dropdown
  overlay={
    <Menu selectable>
      <Menu.Item key="1">1st menu item</Menu.Item>
      <Menu.Item key="2" disabled>
        2nd menu item (disabled)
      </Menu.Item>
      <Menu.Divider />
      <Menu.Item key="3" danger>
        a danger item
      </Menu.Item>
    </Menu>
  }
>
  <Button type="text" icon={<Icon type="DownOutlined" />}>
    Hover me
  </Button>
</Dropdown>

Group

<Dropdown
  overlay={
    <Menu selectable>
      <Menu.ItemGroup title="Group title">
        <Menu.Item>1st menu item</Menu.Item>
        <Menu.Item>2nd menu item</Menu.Item>
      </Menu.ItemGroup>
      <Menu.ItemGroup title="Group title2">
        <Menu.Item>3rd menu item</Menu.Item>
      </Menu.ItemGroup>
    </Menu>
  }
>
  <Button type="text" icon={<Icon type="DownOutlined" />}>
    Hover me
  </Button>
</Dropdown>
