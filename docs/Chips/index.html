Chips 输入框

通过鼠标或键盘输入内容，是最基础的表单域的包装。

何时使用

需要用户输入表单域内容时,还可以进行大小选择。

基本用法

类型

<Chips type="outline" width={200}>
  Content
</Chips>
<Chips type="filled" width={200}>
  Content
</Chips>

Size

<Chips type="outline" width={200}>
  Content
</Chips>
<Chips type="filled" width={200}>
  Content
</Chips>
<Chips type="outline" size="s" width={200}>
  Content
</Chips>
<Chips type="filled" size="s" width={200}>
  Content
</Chips>

前缀图标和后缀图标

<Chips
  type="outline"
  width={200}
  prefix={<Icon type="service/Port" />}
  suffix={<Icon type="place/Agra" />}
>
  Content
</Chips>
<Chips
  type="filled"
  width={200}
  prefix={<Icon type="service/Port" />}
  suffix={<Icon type="place/Agra" />}
>
  Content
</Chips>

<Chips
  type="outline"
  width={200}

  prefix={<Icon type="service/Port" />}

  Content

<Chips

API

属性说明类型默认值
defaultActive默认是否被命中booleanfalse
active是否被命中boolean-
onActiveChange命中变化时的回调(value:boolean)=>void-
type设置类型'outline' | 'filled''outline'
size设置大小's' | 'm''m'
prefix设置前缀图标Icon-
suffix设置后缀图标Icon-
disabled设置失效状态booleanfalse
readonly设置只读状态booleanfalse
loading设置 loading 状态booleanfalse
error设置 error 状态booleanfalse
sendChildrenProps设置是否把一些属性传给 Childrenbooleanfalse
width设置宽度number-
height设置高度（比 size 优先级高）number-
className自定义 CSS classNamestring-
style自定义内联样式CSSProperties-
