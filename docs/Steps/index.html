Steps 步骤

需要显示操作步骤进度时使用

何时使用

基本用法

<Steps
  currentStep={1}
  steps={[
    { name: 'Server Configuration' },
    { name: 'Network Interface' },
    { name: 'Add-on' },
    { name: 'Check-out', disabled: true },
  ]}
/>

API

Steps

属性说明类型默认值

onChange变化回调(index: number, data: StepItem) => void-
